Text Editor Demo

To use or test, include the script text-editor.js and provide a div as an argument to its get_text_editor funtion.

For example-

        <div class="editor"></div>
        <script src="Vanilla-text-editor.js"></script>
        <script>
            window.onload = function() {
                get_text_editor(document.querySelector(".editor"));
            }
        </script>
